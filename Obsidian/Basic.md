### 2/ Style
- Heading 1: # x1
- Heading 2: ## x2
- Heading3: ### x3
- Bold:
	+ How to: **
	+ Shortcut: ctrl + b
	+ Example: **This is bold text**
- Italic:
	+ How to: *
	+ Shortcut: ctrl + i
	+ Example: *This is italic text*
- Highlighter: 
	+ ==
	+ Example: ==This is some highlight text==
- Quote:
	- How to: >
	- Example:
> This is some quotes
- Divider:
	+ How to: ---
---
